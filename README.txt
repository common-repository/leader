LEADER is a Free smart lead capture manager and a set of great out of the box marketing tools.

== Description ==

SMART LEAD CAPTURE
1) Capture lead automatically from any form on your website.
2) Mange leads communication right from LeaderBox (mini CRM).
3) Automatic welcome email
4) Social signature

Leader is a new free WordPress plugin that supercharged your WordPress site with a smart lead capture manager.

= Multilingual & RTL supported. =

Leader supports both RTL and LTR and has the option to be translated by developers to any language.

== Installation ==

= Minimum Requirements =

* WordPress 4.2 or greater
* PHP version 5.2.17 or greater
* MySQL version 5.0 or greater

= Recommended Requirements: =

* WordPress 5.0.3 or greater
* PHP version 7.0 or greater
* MySQL version 5.5 or greater

From WordPress repository:
1. Go to Plugins -> Add New
2. Search for 'Leader'
3. Click the 'Install Now' button
4. Activate the plugin through the 'Plugins' menu
5. Thats it:-).

From file(zip)
1. Upload `leader.zip` in the add new plug page
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Thats it:-).

== Frequently Asked Questions ==

** Is Leader free? **

YES, Leader is free with no contacts (a.k.a leaders) limit and no user limits.

** How do I set it up? **

Leader is a plug&play plug-in, after activation it will automatically capture all contact forms in your website and create a new leader in your system.

** Where does Leader send the email to? **

Leader will send emails to the current admin that is logged and as CC to the default email of your site.

** I’m not receiving emails from Leader? **

WordPress uses virtual email address and it might be in your spam folder, set up SMTP or any other service such as Google app or office 365 and you will receive the email in your inbox.
